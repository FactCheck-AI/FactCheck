Language codes in Dyspatch follow the IETF Language Tag standard which consists of a language code and a country code separated by a hyphen.

For Example

The code for English (United States) 'en-US'. The prefix, 'en', is a language code following the ISO 639-1 standard. The suffix, 'US', is a country code following the ISO 3166-1 Alpha-2 standard.

Supported languages

Dyspatch currently supports 185 different languages.
