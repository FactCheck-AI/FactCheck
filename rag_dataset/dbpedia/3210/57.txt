ARCHIVED: What is Windows Server 2003 R2?

Windows Server 2003 R2 is Microsoft's latest update to Windows Server 2003. R2 provides increased security, management features, and interoperability with other operating systems. R2 is not a service pack; rather, it is a server operating system.

Windows Server 2003 R2 is not available as a free download from Microsoft; however, a 180-day trial version is available for download from Microsoft's TechNet Evaluation Center.
